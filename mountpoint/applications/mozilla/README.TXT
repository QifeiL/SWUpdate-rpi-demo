You may obtain the source code for Firefox from this location via Mozilla's HTTP Servers:

http://releases.mozilla.org/pub/mozilla.org/firefox/releases/latest-3.0/source/

To use the files included in this source code package, copy over contents of the package into your mozilla source code tree. Ignore the overwrite prompts and say yes to the overwrite.

To compile the file, refer to Mozilla's documentation on compiling Firefox. That documentation can be found here:

http://developer.mozilla.org/en/docs/Windows_Build_Prerequisites

NOTE: This link above only describes on how to build IceWeasel and/or Firefox for Windows, but the source code can be applied for all other platforms.
